About

This project uses a financial API from polygon.io, the api is free, so it only allows 5 API calls per minute.
(After doing more than 5 api calls, an object won't be returned).

In this project daily stock and crypto data are displayed, stock data from weekends, holidays or the future cannot be displayed.

Cypto data is a bit thin on some of the currencies and many times the data will not be available.

Play around with the project and have fun.
